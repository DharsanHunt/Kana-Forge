～ながら (nagara) N4

While doing; as; at the same time

expand_less

Construction

Verb-ますstem + ながら

Example Sentences

音楽を聞きながら、勉強します。

Ongaku o kikinagara, benkyou shimasu.

I study while listening to music.

歩きながら電話をしないでください。

Arukinagara denwa o shinaide kudasai.

Please don't talk on the phone while walking.

Usage Notes

Used when the same subject performs two actions simultaneously. The second action mentioned is usually the main focus or primary action.
